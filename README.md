# Smart Monitoring System for Elderly and Solitary Living

## 📌 Project Description
This project is designed to monitor elderly or solitary living persons using ESP32, PIR sensor and Ultrasonic sensor.  
The system checks movement during a set time and automatically sends SOS alert if no movement is detected.

## ⚙️ Components Used
- ESP32
- PIR Sensor
- Ultrasonic Sensor
- Buzzer
- Mobile App (MIT App Inventor)
- WiFi Module (inbuilt ESP32)

## 🚀 Features
- Wake-up time monitoring
- Motion detection
- Auto SOS alert
- Mobile app control
- Safety monitoring system
